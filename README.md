# Fantasy Devs - Frontend

Welcome to the **Fantasy Devs** website! This is a front-end application built using **Next.js** that allows users to build and manage their dream developer teams. Users can log in, sign up, and select developers based on their skills and roles.

## Features

- **User Authentication**: Login and sign-up with Google OAuth 2.0.
- **Team Building**: Users can select developers to form their own fantasy dev teams.
- **Responsive Design**: The website is mobile-friendly and works on all screen sizes.
- **Interactive UI**: Dynamic user interface with smooth transitions and interactive elements.

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Authentication**: Google OAuth 2.0
- **State Management**: React Context API
- **Styling**: Tailwind CSS
- **API**: Communicates with the backend API (you can replace this with your own backend setup)
